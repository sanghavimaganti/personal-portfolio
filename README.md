# Personal Portfolio Website

This is a Full Stack Personal Portfolio Website built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Features
- Responsive portfolio design
- About section
- Skills section
- Projects section
- Contact form
- Frontend and backend integration

## Technologies Used
Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js
